# 📘 Python 期末專題企劃書

## 📌 專題名稱

**記帳系統與支出圖表可視化**

---

## 🎯 專題目的

本專題旨在協助使用者有效記錄每日收支，並透過圖表分析消費趨勢，達成良好的理財管理。系統具備記帳、查詢與可視化分析等功能，以 Python 完成資料處理與視覺化展示。

---

## 🧩 專題核心功能

### 1. 記帳功能

- 新增支出紀錄（日期、分類、金額、備註）
- 查看所有紀錄（列表方式呈現）
- 刪除單筆紀錄（進階選項）

### 2. 查詢功能

- 依月份查詢
- 依類別查詢
- 支出排序（由大到小）

### 3. 統計功能

- 計算指定月份支出總額
- 各分類消費占比統計

### 4. 圖表功能（加分項）

- 類別支出圓餅圖
- 月支出折線圖
- 月份比較長條圖
- 使用 `matplotlib` 或 `pandas` 產生

---

## 🗂️ 使用資料格式

### 資料儲存方式：CSV

檔名：`records.csv`

| 日期 | 類別 | 金額 | 備註 |
| ---- | ---- | ---- | ---- |
| 2025-01-03 | 食物 | 120 | 早餐 |
| 2025-01-03 | 交通 | 30  | 公車 |

---

## 🏗️ 系統架構描述

### 程式主要模組：

- `add_record()`：新增紀錄
- `read_records()`：讀取資料
- `search_by_month()`：依月份查詢
- `summary_category()`：分類統計
- `plot_pie()`：圓餅圖
- `plot_line()`：折線圖
- `main()`：主選單流程

---

## 🔄 系統流程圖

```text
使用者選擇功能
   │
   ├── 新增紀錄 → 儲存至 CSV
   │
   ├── 查詢紀錄 → 顯示表格
   │
   ├── 每月統計 → 計算總額
   │
   ├── 圖表分析 → 產生可視化圖
   │
   └── 結束程式
```

## 🛠️ 使用技術

- Python 3
- `csv` 模組處理資料
- `matplotlib` 繪製圖表
- `datetime` 處理時間格式
